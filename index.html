Red Hat Certified Specialist in OpenShift Administration OpenShift Administration

The Red Hat Certified Specialist in OpenShift Administration exam (EX280) assesses your ability to design, implement, and manage a cloud application platform with Red Hat速 OpenShift速 Container Platform.

You become a Red Hat Certified Specialist (RHCS速) after passing this test, and it also counts toward becoming a Red Hat Certified Architect (RHCA速).

This exam uses Red Hat OpenShift Container Platform 4.6 as its foundation.

Prerequisites:

  • Become a Red Hat Certified System Administrator (RHCSA) or have professional experience and skills that are comparable.
  • Take the Red Hat OpenShift Administration II: Operating a Production Kubernetes Cluster (DO280) course or have equivalent OpenShift Container Platform job experience.
  • Exam objectives for Red Hat Certified Specialist in OpenShift Administration (EX280)
  • It is recommended that you have prior experience with container technology.
  • Take our free assessment to determine which course will best complement your exam preparation.

Objectives:

  • OpenShift Container Platform administration.
  • Users and policies can be managed.
  • Access to resources can be restricted.
  • Set up the networking components.
  • Create and administer software.
  • Set up pod scheduling.
  • Set the cluster scaling parameters.
